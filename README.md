# Eagle Concrete Coatings LLC

Professional concrete coating website built with Next.js 15, Tailwind CSS, and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Red, navy, and white color scheme
- Interactive before/after image slider
- Floating call-now button (instant phone call)
- Separate pages for Colors & Finishes and Gallery
- Interactive Google Maps service area
- Mobile-responsive with working hamburger menu
- Customer reviews and quote form
- Trust badges: locally owned, licensed & insured, years of experience
